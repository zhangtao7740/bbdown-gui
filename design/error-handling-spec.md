# 错误处理规范

## 目标

所有错误都必须告诉用户三件事：

1. 发生了什么。
2. 为什么可能发生。
3. 下一步怎么恢复。

## 错误展示层级

| 层级 | 使用场景 | 形式 |
| --- | --- | --- |
| Inline | 表单校验、路径为空、选项冲突 | 字段下方错误 |
| Banner | 页面级可恢复错误 | 顶部提示条 |
| Dialog | 破坏性确认或需要用户立即决策 | 弹窗 |
| Toast | 操作结果反馈 | 右下角短提示 |
| Log | 详细排查信息 | 队列日志/诊断日志 |

## 通用错误文案结构

```text
标题：无法开始下载
说明：未检测到 BBDown 可执行文件。
建议：请在设置页选择 BBDown 路径，或将 BBDown 放到应用同级目录后重新检测。
操作：去设置 / 重新检测
```

## 下载页错误

| 场景 | 文案 | 恢复操作 |
| --- | --- | --- |
| 链接为空 | 请先输入 Bilibili 链接或 BV 号。 | 聚焦输入框 |
| 链接格式不支持 | 无法识别这个链接。请确认它是 Bilibili 视频链接或 BV 号。 | 修改链接 |
| 解析超时 | 解析超时，可能是网络较慢或接口暂时不可用。 | 重试解析 |
| 未检测到 BBDown | 未检测到 BBDown，无法解析或下载。 | 去设置 / 重新检测 |
| 未选择分 P | 至少选择一个分 P 后才能下载。 | 选择分 P |
| 保存目录为空 | 请先选择保存目录。 | 浏览目录 |
| 保存目录不可写 | 当前保存目录不可写。 | 更换目录 |
| 仅音频/仅视频冲突 | 仅音频和仅视频不能同时启用。 | 切换内容模式 |
| aria2c 未检测到 | 已启用 aria2c，但未检测到 aria2c。 | 关闭 aria2c / 去设置 |

## 队列页错误

| 场景 | 文案 | 恢复操作 |
| --- | --- | --- |
| 网络中断 | 下载中断，正在等待重试。 | 暂停 / 立即重试 |
| 登录态失效 | 当前账号登录态可能已失效。 | 去登录 |
| 清晰度无权限 | 当前账号无权下载所选清晰度。 | 降低清晰度 / 去登录 |
| FFmpeg 合并失败 | FFmpeg 合并失败，文件可能已下载但未合并。 | 查看日志 / 重新后处理 |
| 磁盘空间不足 | 磁盘空间不足，任务已暂停。 | 更换目录 / 清理空间 |
| 任务被取消 | 任务已取消，不会继续下载。 | 重新加入队列 |

## 历史页错误

| 场景 | 文案 | 恢复操作 |
| --- | --- | --- |
| 文件缺失 | 找不到这个产物文件。 | 重新定位 / 移除记录 |
| 重命名失败 | 重命名失败，可能是文件正在使用或名称非法。 | 修改名称 / 关闭占用程序 |
| 移动失败 | 移动失败，目标目录可能不可写。 | 更换目录 |
| 后处理失败 | 后处理失败，请查看 FFmpeg 日志。 | 查看日志 / 重试 |
| 媒体扫描失败 | 无法读取媒体信息。 | 重新扫描 / 忽略 |

## 设置页错误

| 场景 | 文案 | 恢复操作 |
| --- | --- | --- |
| 工具路径不存在 | 这个路径不存在。 | 重新选择 |
| 工具不可执行 | 文件存在，但无法作为可执行程序运行。 | 重新选择 |
| 工具版本读取失败 | 已找到工具，但无法读取版本。 | 查看诊断 / 重新检测 |
| 登录二维码获取失败 | 无法获取登录二维码。 | 重试 / 检查 BBDown |
| 登录取消 | 登录已取消。 | 重新登录 |
| 保存设置失败 | 设置保存失败，可能没有写入权限。 | 重试 |

## Error Boundary

React 渲染错误展示：

- 标题：界面渲染出错
- 说明：应用界面遇到异常，下载任务不一定受影响。
- 操作：
  - 重新加载界面
  - 复制诊断信息
  - 打开日志目录

要求：
- 不显示乱码。
- 不暴露 Cookie。
- 复制诊断信息时脱敏路径中的用户名可选，但必须脱敏 Cookie。
