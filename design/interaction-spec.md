# 交互规范

## 范围决策

本规范只覆盖需要补齐的桌面交互：

- 右键菜单
- 快捷键
- 焦点流转
- 响应式和最小窗口

明确不做：

- 不做拖拽链接
- 不做拖拽文件
- 不做新手引导

## 右键菜单

### 下载页

链接输入框：
- 粘贴
- 清空
- 解析

视频预览区：
- 复制标题
- 复制 BV 号
- 复制原始链接

分 P 列表：
- 选择此分 P
- 取消选择此分 P
- 全选
- 反选

右侧最近任务：
- 打开所在目录
- 查看历史记录
- 复制任务标题

### 队列页

任务组：
- 暂停任务组
- 继续任务组
- 重试失败子任务
- 打开保存目录
- 复制标题
- 复制 BV 号
- 查看日志

子任务：
- 暂停
- 继续
- 重试
- 取消
- 打开输出目录
- 复制错误信息
- 复制日志

日志面板：
- 复制当前行
- 复制全部日志
- 只看错误
- 清空当前显示，不删除持久日志

### 历史页

历史任务：
- 打开任务目录
- 复制标题
- 复制 BV 号
- 重新扫描目录
- 删除历史记录，不删除文件

产物行：
- 打开文件
- 打开所在目录
- 重命名
- 移动
- 重新定位
- 后处理
- 移除记录，不删除文件

### 设置页

工具路径输入：
- 粘贴
- 清空
- 选择文件
- 重新检测

账号区：
- 扫码登录
- 刷新状态
- 退出登录

## 焦点流转

### App Shell

Tab 顺序：

1. 标题栏可操作按钮
2. 侧栏导航
3. 页面主操作区
4. 页面次操作区
5. 底部状态栏可操作项

侧栏导航：
- 上下方向键切换导航项。
- Enter 激活导航项。

### 弹窗

- 打开弹窗后焦点进入弹窗标题后的第一个可操作控件。
- Tab 在弹窗内部循环。
- Esc 关闭非破坏性弹窗。
- 破坏性操作必须先聚焦取消按钮，不能默认聚焦确认。

### 表格和列表

- 历史产物表支持行级聚焦。
- Enter 展开或触发主操作。
- 右键菜单打开后焦点进入菜单第一项。
- 菜单关闭后焦点回到触发元素。

## 响应式和最小窗口

最小窗口：
- 推荐最小宽度：1100px
- 推荐最小高度：720px
- 硬性最小宽度：960px
- 硬性最小高度：640px

断点：

| 宽度 | 布局 |
| --- | --- |
| >= 1280px | 左侧导航 + 主工作区 + 右侧状态栏 |
| 1100-1279px | 左侧导航 + 主工作区，右侧状态栏下移 |
| 960-1099px | 左侧导航变窄，右侧状态栏隐藏为页面内折叠区 |

规则：
- 主下载选项区不能横向溢出。
- 历史产物表在窄窗口允许横向滚动。
- 任务日志限制高度，不能撑破页面。
- 顶部输入区按钮在窄窗口下允许换行，但主按钮保持可见。
